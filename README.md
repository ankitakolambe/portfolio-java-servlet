# Ankita Kolambe | Portfolio Website

**Portfolio Website of Ankita Kolambe – Java Full Stack Developer**

This is a personal portfolio website built to showcase my skills, experience, and projects as a **Java Full Stack Developer**. The website features a clean and modern design with responsive layouts and smooth scrolling animations.

## Features

* **Hero Section**: Engaging introduction with name, role, and quick links to projects and resume.
* **About Section**: Overview of skills, technologies, and development experience.
* **Experience Timeline**: Highlights internships, student club activities, and professional experiences.
* **Resume Section**: Downloadable resume link for recruiters and collaborators.
* **Projects Section**: Showcase of projects with descriptions, key features, tech stack, and GitHub links.
* **Contact Form**: Allows visitors to send messages directly via email.
* **Responsive Design**: Works seamlessly across desktop, tablet, and mobile devices.
* **Animations**: Smooth scrolling and entry animations using Intersection Observer API.
* **Footer with Social Links**: Quick access to LinkedIn, GitHub, and Email.

## Tech Stack

* **Frontend**: HTML5, CSS3, JavaScript
* **Animations**: CSS animations & Intersection Observer API
* **Backend (for contact form)**: Can be connected to any server-side language (currently placeholder)
* **Projects Showcased**:

  * Dolphin Cruises – Responsive static website
  * SketchMatch – AI-powered forensic sketch matching system
  * Online Banking System – Java Swing & JDBC application

## How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/ankitakolambe/Portfolio.git
   ```
2. Open `index.html` in your browser.
3. Update any content, projects, or contact form settings as needed.
